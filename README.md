# Landing Page Institucional

Esta é uma landing page moderna em HTML/CSS/JS com foco em performance e visual contemporâneo. A **segunda seção** é um **carrossel de imagens** (portfólio), conforme solicitado.

## Como usar
- Abra o arquivo `index.html` diretamente no seu navegador.
- Não há dependências externas de build. Apenas HTML/CSS/JS puros.

## Personalização
- Imagens do carrossel: edite os `src` dentro da seção `#portfolio` no `index.html`.
- Cores e temas: ajuste as variáveis CSS em `styles.css` (bloco `:root`).
- Conteúdo: substitua textos de seções `#hero`, `#sobre`, `#servicos` e `#contato` conforme necessidade.

## Recursos inclusos
- Header com navegação e comportamento para mobile.
- Seção Hero com elementos decorativos.
- Carrossel responsivo com autoplay, setas, dots, teclado e swipe.
- Seções Sobre, Serviços e CTA de contato.
- Rodapé com âncora de voltar ao topo.

## Observações
- As imagens de exemplo usam `picsum.photos` (placeholders). Com internet elas serão exibidas normalmente.
